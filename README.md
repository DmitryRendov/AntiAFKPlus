# AntiAFK+ / Detect every AFK-Form [RECODED]

This Plugin detects every AFK Player.
You can perform customized Actions for AFK Players.
And punish them like you want.

### Features 

- Detects a lot AFK-Machines
- Run every Command you want
- Play Sounds
- Send Messages
- Everything customizable

### Commands 

/AntiAFK reload - Reload the Plugin

### Permissions

AntiAFKPlus.bypass - Bypass the AFK Check
AntiAFKPlus.reload - Reload the AFK Plugin
